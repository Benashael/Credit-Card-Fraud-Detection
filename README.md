# Credit-Card-Fraud-Detection
Building a machine learning model to detect credit card frauds using Scikit-Learn's Decision Tree and SVM.
## Package Used
1. Matplotlib
2. Pandas
3. Pylab
4. Numpy
5. Os
6. Sklearn
7. Opendatasets
8. Snap ML (Optional)
## Dataset
Dataset can be accessed from the following link:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
## Execution
The execution of this model can be done on both Jupyter notebook and Google Colab.
